# StatikManager

WPF-Tool zur Anzeige, Analyse und zum Zuschneiden von PDF- und Word-Dokumenten mit optimierter Performance und moderner Zoom- und Navigationslogik.

## Funktionen

- Anzeige von PDF- und Word-Dokumenten
- Automatische Rand-Erkennung (Beschnitt)
- Mehrseitenansicht (vertikal & horizontal)
- Smooth Zoom mit Mausfokus
- Stabile Lade- und Rendering-Logik
- Mehrfachauswahl und Löschen von Dateien

## Technologie

- C#
- WPF (.NET Framework)
- PDFium (Docnet)
